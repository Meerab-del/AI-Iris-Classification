# AI-Iris-Classification
#  AL-2002 Project — Module 4 (Learning in AI)

##  Overview
This project implements and compares fundamental machine learning algorithms including **Perceptron Learning Rule** and **Gradient Descent Delta Rule** using the **Iris dataset**.

It demonstrates how different learning strategies behave on a binary classification problem.

---

##  Features
- Perceptron Learning Algorithm
- Gradient Descent Delta Rule (Linear & Sigmoid)
- Iris Dataset preprocessing (binary classification)
- Data normalization and train-test split
- Accuracy evaluation and learning curves

---

##  Concepts Used
- Perceptron Learning Rule  
- Gradient Descent Optimization  
- Activation Functions (Step, Linear, Sigmoid)  
- Supervised Learning  
- Binary Classification  
- Feature Scaling (StandardScaler)

---

##  Project Structure
- `main.py` → Full implementation of models  
- Dataset → Iris dataset (via sklearn)  
- Outputs → Accuracy results & graphs
